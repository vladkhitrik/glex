# Glex

Glex is a light-blue themed website about rockets, boats, and drones.

## Features

- Dark-blue `GLEX` header and light-blue UI palette
- NASA-style rocket blueprint background layer
- Foreground rocket takeoff visual style
- Top `Info` button with a full list of website capabilities
- Toolbar sections:
  - Learn Rockets
  - Learn Boats
  - Learn Drones
  - Credits
  - Profile
- Credits panel with:
  - Daniil Khitrik
  - Oscar Rushrook
  - AI
- Profile customization:
  - Change display name
  - Upload profile image
  - Saved automatically in browser local storage

## Run Locally

Because Node.js install was canceled on this machine, this project is set up as a modern static web app (HTML/CSS/JavaScript modules) with no build step required.

Use any static server, for example:

```powershell
python -m http.server 5500
```

Then open `http://localhost:5500`.
